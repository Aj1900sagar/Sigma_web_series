What is the semantic tag?

A semantic tag is an HTML tag that conveys meaning about the content contained within it. These tags help search engines and other user agents understand the structure and importance of the content on a webpage.

Examples of semantic tags include:

  • <header> - Represents the header of a section or page.
  • <footer> - Represents the footer of a section or page.
  • <article> - Represents a self-contained piece of content.
  • <section> - Represents a thematic grouping of content.
  • <nav> - Represents a navigation menu.
  • <aside> - Represents content that is tangentially related to the content around it.
  • <main> - Represents the main content of the document.
  • <figure> - Represents self-contained content, often with a caption.
  • <figcaption> - Represents a caption for a <figure> element.
  • <mark> - Represents highlighted text.
  • <time> - Represents a specific time or date.
  • <details> - Represents additional details that the user can view or hide.
  • <summary> - Represents a summary or caption for a <details> element.

Using semantic tags can improve the accessibility and SEO of a webpage.
